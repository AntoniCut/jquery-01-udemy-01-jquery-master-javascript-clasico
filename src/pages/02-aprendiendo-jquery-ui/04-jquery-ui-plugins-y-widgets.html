Tooltips



Los tooltips son cuadros de texto que aparecen al situar el cursor sobre un elemento. Generalmente se usan para dar información adicional sobre un elemento.

Dialogos



Los cuadros de dialogo son ventanas emergentes que se usan para mostrar información o pedir confirmación al usuario.

Calendarios



Los calendarios permiten seleccionar fechas de manera visual y sencilla. Son útiles para formularios y aplicaciones que requieren entrada de fechas.

Pestañas



Las pestañas permiten organizar contenido en secciones que se pueden mostrar u ocultar. Facilitan la navegación y mejoran la experiencia del usuario.
